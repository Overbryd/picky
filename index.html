The Ruby semantic text search engine.

Picky is a really easy to use and fast Ruby search engine that helps your users find what they are looking for.

Try…

  • A simple library example (wait 3 seconds for it to boot up).
  • A telephone book search for Switzerland: TwixTel online.

What it is

A semantic text search engine for categorized data, such as varchar fields from a database (probably like most – if not all – data in your application).

The server offers a HTTP interface, and a comfortable web front end for your application. There's also a statistics web app and a live interface for changing parameters inside the running server.

What it is not

A full-text search engine for uncategorized data (like text from a book). If you need that, look at Sphinx or Solr (Lucene). They will help.

Where it is

Picky lives in a little cave called The Github Repo. He usually pays his rent on time.

Getting started: Got 5 minutes? Install and run Picky on your computer and adapt the examples.
